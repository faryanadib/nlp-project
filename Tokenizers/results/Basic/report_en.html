Overview

This session built on the from-scratch classifiers (Naive Bayes, Perceptron, Logistic Regression) developed in Weeks 1–5. We explored two directions promised in the midway presentation: better feature representations using TF-IDF and bigrams, and pre-trained language model embeddings using BERT variants.

All models are evaluated on the same 80/20 train-test split (seed=42) of the IMDB dataset using Accuracy, Precision, Recall, and F1.

Key Findings

1

Generic BERT embeddings underperform TF-IDF: Without any sentiment adaptation, the [CLS] token from distilbert-base-uncased encodes general language structure rather than sentiment signals, resulting in F1=0.8614 — below even the from-scratch LR baseline.

2

Domain adaptation matters: Moving from a generic model → SST-2 fine-tuned → IMDB fine-tuned produced clear stepwise improvements (0.8614 → 0.8917 → 0.9288 → 0.9545). Each step brought the embedding space closer to the task at hand.

3

SVM vs LR: LinearSVC consistently outperformed Logistic Regression on TF-IDF features. With BERT embeddings, LR performed better — suggesting the embedding space is more LR-friendly (smoother, more linearly separable) than raw TF-IDF space.
